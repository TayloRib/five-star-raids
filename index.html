As if Herba Mystica wasn't hard enough to find, only certain pokémon will drop every type of Herba. If you're low on a variety of types in general it makes sense to start with some raids that have a possibility of droping every type (if you can find them).

Otherwise if you're low on a certain type of Herba it might make more sense to start with a raid that drops a single type as those raids are a lot more common.
